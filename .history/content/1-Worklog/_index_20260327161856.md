# Worklog

This Worklog page details my entire internship and working process during the 11-week First Cloud AI Journey (FCAJ) program at AWS. Below is a summary of my learning path, Cloud architecture research, and the practical implementation of the SmartInvoice project week by week:

* **Week 1:** Familiarize with AWS (EC2, S3, VPC, RDS), practice account management (Budgets, Alerts), and attend the FCAJ Kickoff.
* **Week 2:** Deep dive into AWS IAM, VPC network architecture (Subnet, Route Table, Gateways), and AWS Well-Architected Framework.
* **Week 3:** Practice configuring VPN, Direct Connect, Load Balancer (ELB), Route 53, VPC Peering, and Transit Gateway.
* **Week 4:** Manage EC2 resources (AMI, EBS, EFS), practice Auto Scaling, CloudWatch, and attend the AWS re:Invent Recap HCMC event.
* **Week 5:** Explore storage solutions (S3, Storage Gateway), Disaster Recovery architecture, AWS Backup, and CloudFront.
* **Week 6:** Focus on security deployment: AWS KMS, Security Hub, Cognito, IAM Permission Boundary, and AWS Organizations.
* **Week 7:** Learn RDS database, configure JWT Token security, and handle XML invoice data normalization.
* **Week 8:** Build RBAC authorization feature, Audit Logs system, and invoice management module (CRUD, Filtering, Pagination).
* **Week 9:** Develop Subscription feature, multi-level approval workflow, and integrate VNPay, VietQR payments.
* **Week 10:** Apply AI/ML: Configure GPU Docker, integrate OCR engine for image extraction, and Gemini API for data processing.
* **Week 11:** Optimize performance, fix storage leak issues, finalize the user system, and merge source code.
