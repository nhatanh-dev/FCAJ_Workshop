Trang Worklog này ghi chú lại chi tiết toàn bộ quá trình thực tập và làm việc của mình trong suốt 12 tuần tham gia chương trình First Cloud AI Journey (FCAJ) tại AWS. Dưới đây là tóm tắt lộ trình học tập, nghiên cứu kiến trúc Cloud và quá trình triển khai thực tế dự án SmartInvoice của mình qua từng tuần:

**Tuần 1:** [Làm quen với AWS (EC2, S3, VPC, RDS), thực hành quản lý tài khoản (Budgets, Alerts) và tham gia Kickoff FCAJ.](1.1-week1/)

**Tuần 2:** [Tìm hiểu chuyên sâu về AWS IAM, kiến trúc mạng VPC (Subnet, Route Table, Gateways) và AWS Well-Architected Framework.](1.2-week2/)

**Tuần 3:** [Thực hành cấu hình VPN, Direct Connect, Load Balancer (ELB), Route 53, VPC Peering và Transit Gateway.](1.3-week3/)

**Tuần 4:** [Quản lý tài nguyên EC2 (AMI, EBS, EFS), thực hành Auto Scaling, CloudWatch và tham gia event AWS re:Invent Recap HCMC.](1.4-week4/)

**Tuần 5:** [Tìm hiểu các giải pháp lưu trữ (S3, Storage Gateway), kiến trúc Disaster Recovery, AWS Backup và CloudFront.](1.5-week5/)

**Tuần 6:** [Tập trung triển khai bảo mật: AWS KMS, Security Hub, Cognito, IAM Permission Boundary và AWS Organizations.](1.6-week6/)

**Tuần 7:** [Tìm hiểu cơ sở dữ liệu RDS, cấu hình bảo mật JWT Token và xử lý chuẩn hóa dữ liệu hóa đơn XML.](1.7-week7/)

**Tuần 8:** [Xây dựng tính năng phân quyền RBAC, hệ thống Audit Logs và module quản lý hóa đơn (CRUD, Filtering, Pagination).](1.8-week8/)

**Tuần 9:** [Phát triển tính năng gói cước (Subscription), luồng phê duyệt đa cấp và tích hợp thanh toán VNPay, VietQR.](1.9-week9/)

**Tuần 10:** [Ứng dụng AI/ML: Cấu hình Docker GPU, tích hợp engine OCR bóc tách ảnh và Gemini API cho xử lý dữ liệu.](1.10-week10/)

**Tuần 11:** [Tối ưu hiệu năng, xử lý lỗi rò rỉ dung lượng, hoàn thiện hệ thống người dùng và hợp nhất (merge) mã nguồn.](1.11-week11/)
